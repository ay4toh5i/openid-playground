# oidc-playground

OpenID Connect / OAuth2.0 playground built with Deno + Honox. Uses the Honox
islands architecture with the React renderer.

## Structure

```
app/
  server.ts
  global.d.ts
  routes/
```

## Run locally

```bash
deno task dev
```

## Build & preview

```bash
deno task build
deno task preview
```
